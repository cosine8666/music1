# 音樂應用程式

一個互動式的音樂學習應用程式，使用 Flutter 開發。

## 功能特色

- 🎵 互動式鋼琴鍵盤
- 🎼 樂譜顯示
- ⏱️ 30秒計時挑戰
- 🔊 音效回饋
- 📱 響應式設計

## 遊戲玩法

1. 點擊 "Start" 開始遊戲
2. 觀察樂譜顯示的音符
3. 在鋼琴鍵盤上點擊對應的琴鍵
4. 在30秒內盡可能答對更多題目
5. 查看最終成績

## 技術架構

- **框架**: Flutter
- **音效**: audioplayers
- **部署**: GitHub Pages

## 本地開發

### 需求
- Flutter SDK
- Dart SDK

### 安裝步驟

1. 克隆專案
```bash
git clone https://github.com/cosine8666/music1.git
cd music1
```

2. 安裝依賴
```bash
flutter pub get
```

3. 執行應用程式
```bash
flutter run
```

### 建置 Web 版本
```bash
flutter build web
```
